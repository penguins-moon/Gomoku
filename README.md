# Gomoku
python实现的五子棋小游戏， 使用PySide6搭建图形化用户界面，实现了alpha-beta prunning 和 MCTS 算法，v2版本基于MCTS参考ALPHAZERO的思路实现深度强化学习训练策略价值网络

# 环境依赖

- Python version = 3.8
- PySide6 version = 6.6.2
- qtmodern
- Pytorch version = 2.2.2
- NumPy version = 1.24.3
- cuda version >= 12.1
